# Common Keys of Top-Level Specs in Vega-Lite

The following keys are common to all top-level specifications in Vega-Lite
(e.g., Unit, Layer, Facet, Concat). These are defined in `TopLevelProps`.

- `$schema`
- `autosize`
- `background`
- `config`
- `data`
- `datasets`
- `description`
- `name`
- `padding`
- `params`
- `title`
- `transform`
- `usermeta`
